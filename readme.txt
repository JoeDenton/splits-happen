SplitsHappen 
Asks the user to input a string representing rolls in a game of bowling, and calculates the game total 


Running the main program: 
Compile SplitsHappen.java from command line using Java or in IDE of your choice
Follow the prompt after compiling the program

Running the test cases: 
Save SplitsHappen.java and SplitsHappenTest.java into a project in an IDE 
Add Junit5 library to the project
Run SplitsHappenTest.java

Built with Java in Eclipse
